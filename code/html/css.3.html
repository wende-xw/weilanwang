  • 苑子豪

    苑子豪，青年作家，演员。当当网年度新锐作家,第十届作家榜金奖·年度新锐作家,中华儿慈会益童成长中心爱心大使,星月榜样盛典年度青年榜样,《鲁豫有约》《天天向上》人气嘉宾。

    相关作品： 我们的十年（签名本 北大双胞胎苑子豪新书） 不好意思，次当大人 我们都一样.年轻又彷徨

  • 曹文轩

    曹文轩，北京大学中文系教授，中国作家协会主席团委员，北京作家协会副主席。 著有长篇小说《山羊不吃天堂草》《草房子》《红瓦》《根鸟》《细米》《青铜葵花》《火印》等。

    相关作品： 草房子（曹文轩经典之作，经典中的经典！） 我们回家 小酒店里的怪客 曹文轩小说馆 回顾大师三十年创作之路，展现一个完整的曹文轩
